Para poder utilizar las funciones del repositorio recomiendo copiar una estructura similar a la presente en los archivos de tests. En el caso de querer graficar el poliedro de un problema de suma constante por filas y columnas usar los scripts dentro de Poliedro3D4D respectivos. Recomiendo usar malla4D para poliedros cuando el problema es cuatridimensional (n-1)*(m-1)=4 y SolNM3D para cuando el problema es tridimensional. De forma automática generan problemas de suma aleatorios, para poner uno en particular escoger f = suma por filas y l = suma por columnas del problema particular, o cambiar V por una matriz solución del sistema (en particular se suele asumir que V es un vértice). La notación es la misma en todos los programas.
